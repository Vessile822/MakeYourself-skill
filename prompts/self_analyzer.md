# 自我記憶分析 Prompt

## 任務

你將收到使用者關於自己的原材料（聊天記錄、日記、照片、口述等）。
從中提取關於 **{name}** 的自我認知與人生軌跡，用於建構 Self Memory。

**原則：只提取與使用者本人強相關的內容，有依據才寫，沒有就標註「原材料不足」。**

---

## 提取維度

### 1. 核心價值觀

從使用者的表達中提取反覆強調的價值判斷：
- 對工作的態度（事業/穩定/自由/意義感）
- 對金錢的態度（累積/消費/無所謂）
- 對人際關係的態度（獨處/親密/廣泛社交）
- 對成長的看法（進取/舒適/隨緣）
- 自由 vs 安全的優先順序
- 長期主義 vs 即時滿足

```
輸出格式：
工作觀：[描述]
金錢觀：[描述]
關係觀：[描述]
成長觀：[描述]
核心衝突：[如果有明顯矛盾，如「渴望自由又害怕不穩定」]
```

### 2. 生活習慣

- 作息規律（早起型/夜貓子/不規律）
- 飲食偏好和模式
- 空間偏好（宅在家/出門/特定場所）
- 消費觀念（節儉/衝動/理性/極簡/囤貨）
- 日常儀式（固定的習慣或儀式感）

```
輸出格式：
作息：[描述]
飲食：[描述]
空間：[描述]
消費：[描述]
儀式感：[描述]
```

### 3. 重要記憶

- 人生關鍵節點（升學、搬家、換工作、分手、重大決定）
- 反覆提及的場景或時期
- 轉折點事件及其影響
- 仍在影響的童年/青少年經歷（如有依據）

```
輸出格式：
關鍵節點：[按時間線列出]
反覆回憶的場景：[列表]
轉折點：[事件 + 影響]
```

### 4. 人際關係圖譜

- 對家人的典型態度（親近/疏離/矛盾）
- 對朋友的態度（重視質/量、維繫方式）
- 對親密關係的態度（渴望/迴避/糾結）
- 處理衝突的典型方式
- 在關係中的常見角色（照顧者/被照顧者/旁觀者）

```
輸出格式：
對家人：[描述 + 典型場景]
對朋友：[描述 + 典型場景]
對親密關係：[描述 + 典型場景]
衝突處理：[描述 + 典型場景]
關係角色：[描述]
```

### 5. 成長軌跡

- 近幾年最明顯的變化
- 正在努力的方向
- 反覆掙扎的課題
- 自我接納程度的變化

```
輸出格式：
近幾年的變化：[描述]
努力方向：[列表]
反覆掙扎的課題：[列表]
自我接納：[描述]
```

---

## 輸出要求

- 語言：繁體中文（台灣用語）
- 沒有資訊的維度：標註 `（原材料不足）`
- 有原文依據的結論：引用原話（加引號）
- 避免過度推斷，尤其不要進行心理學診斷
- 輸出結果直接用於產生 self.md，要求具體可感，不要寫抽象形容詞
