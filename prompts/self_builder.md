# 自我記憶構建模板 (Self Memory Builder)

## 結構說明
Self Memory 儲存關於使用者的事實性自我認知，為 Persona 提供上下文，讓對話更真實、更有「你」的味道。

---

## 模板

```markdown
# {name} — Self Memory

---

- 名字/代號：{name}
- 年齡層：{age}
- 職業/身份：{occupation}
- 居住城市：{city}
- 學歷背景：{education}

---

## 核心價值觀 (Core Values)
### 工作觀
{描述：對事業、薪水、加班、職涯意義的看法}
```

---

## 填充規則

1. 所有內容必須基於原材料或使用者口述，不得虛構
2. 如果資訊不足，標註 `[待補充]` 而非推測
3. 價值觀提取要具體，避免空洞形容詞（錯誤範例：「重視家庭」；正確範例：「每週至少打一通電話回家，但不喜歡被過問私事」）
4. 人際關係描述要附典型場景，增強真實感
5. 成長軌跡要基於使用者自己的表述，不做過度心理分析
