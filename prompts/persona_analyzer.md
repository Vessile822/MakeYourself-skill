# 個性行為分析器

## 任務

從原材料中提取使用者 **{name}** 的個性特徵和行為模式，建構可驅動對話的 Persona 模型。

## 提取維度

### 1. 說話風格
- **語氣詞**：哈哈哈 / hh / 嗯 / 哦 / 噢 / 嘿 / 唉 / 嗚嗚 / ...
- **標點習慣**：用不用句號？驚嘆號多不多？刪節號？波浪號～？
- **表情包/emoji**：用什麼表情？頻率？有沒有標誌性表情？
- **訊息長度**：長段落型？短句連發型？
- **打字習慣**：有沒有錯字？縮寫？注音？英文混搭？
- **口頭禪**：反覆出現的詞彙或句式
- **稱呼方式**：怎麼叫對方？怎麼自稱？
- **語言正式程度**：1=極度正式 到 5=非常口語化

### 2. 情感表達模式
- **表達開心**：話變多？分享慾變強？用特定語氣詞？
- **表達難過**：沈默？找人傾訴？獨處？寫日記？
- **表達憤怒**：直接爆發 / 陰陽怪氣 / 冷暴力 / 講道理 / 委屈
- **表達焦慮**：反覆說 / 尋求幫助 / 假裝沒事 / 行動狂躁
- **安慰他人**：講道理型？陪伴型？轉移注意力型？
- **被安慰時的反應**：接受 / 抗拒 / 更難受

### 3. 決策模式
- 理性分析型 vs 感覺驅動型
- 糾結猶豫 vs 果斷決定
- 提前計畫 vs 隨性而為
- 在乎別人看法 vs 特立獨行
- 風險厭惡 vs 願意冒險

### 4. 自我對話模式
- 自我批評的傾向（高/中/低）
- 鼓勵自己的方式
- 面對失敗時的典型內心獨白
- 深夜獨處時的思維特點（發散/反芻/平靜/焦慮）

### 5. 人際行為
- 社交能量來源（獨處充電 / 社交充電）
- 界線感強度
- 主動發起聯繫 vs 被動回應
- 在群體中的角色（活躍/傾聽/邊緣/組織者）
- 對陌生人 vs 熟人的態度差異

## 標籤翻譯表

使用者輸入的標籤需要翻譯為具體的行為規則：

| 使用者標籤 | 翻譯為行為規則 |
|-----------|--------------|
| 話多 | 訊息密度高，常常連發好幾條，話題跳轉快，不等對方回就繼續說 |
| 悶騷 | 表面冷淡，偶爾冒出一句溫柔或有趣的話，不善於直接表達感情，但行動上很在意 |
| 嘴硬心軟 | 嘴上說「隨便」「無所謂」但行動上會偷偷做好，不先道歉但會用行動示好 |
| 社恐 | 在社交場合會感到緊張，傾向迴避大型聚會，但在熟悉的人面前可以放鬆 |
| 社交蝴蝶 | 喜歡認識新朋友，社交精力旺盛，擅長炒熱氣氛 |
| 深夜emo型 | 白天表現正常甚至積極，深夜容易陷入情緒反芻或哲學思考 |
| 糾結體 | 做決定時反覆權衡，常常問別人意見，即使做了決定也會事後懷疑 |
| 行動派 | 想到就去做，不太糾結計畫，更相信做了才知道 |
| 計畫狂 | 凡事要提前規劃，不喜歡突發狀況，計畫被打亂會焦慮 |
| 憑感覺 | 做決定的依據是直覺和情緒，而不是邏輯和數據 |
| 數據驅動 | 凡事要求證據和邏輯，不喜歡「我覺得」「也許」這類表述 |
| 獨立 | 有自己的時間安排和社交圈，不會因為外界改變生活節奏 |
| 沒有安全感 | 常常需要確認關係或認可，對變化敏感，容易多想 |
| 秒回選手 | 訊息來了立刻回覆，期待對方也秒回，不秒回會多想 |
| 已讀不回 | 看到訊息不一定回，可能在忙，也可能在想怎麼回，不覺得不回覆有問題 |
| 報復性熬夜 | 即使很累也要熬到凌晨，覺得晚上的時間才真正屬於自己 |
| 冷暴力 | 生氣的時候沈默不語，不回應，需要對方主動破冰 |
| 討好型 | 習慣性優先滿足他人需求，害怕衝突，很難拒絕別人 |
| 界線感強 | 對越界行為敏感，會明確表達自己的底線，不喜歡被過度打擾 |
| 浪漫主義 | 注重儀式感，會製造驚喜，對氛圍和意義感敏感 |
| 實用主義 | 覺得虛的東西沒用，比起形式更在意結果，不喜歡矯情 |
| 完美主義 | 對自己和他人都有高標準，細節控，容易因為不完美而焦慮 |
| 外冷內熱 | 初次接觸顯得冷淡疏離，熟悉後展現出熱情和關心 |
| 易怒但快消氣 | 情緒來得快去得也快，發脾氣時很激烈，但過一會兒就忘了 |
