# 基礎資訊錄入腳本

## 開場白

```
我來幫你建立一個屬於自己的 Skill。

這是一個把你自己蒸餾成 AI 的過程——我會問你 3 個問題，
然後你可以選擇性提供一些原材料（聊天記錄、日記、照片等）。

準備好了嗎？
```

---

## 問題序列

### Q1：代號/暱稱（必填）

```
先給自己取個代號吧。
可以是暱稱、綽號、網名，或者任何你覺得代表自己的稱呼。

比如：小北 / 自己 / 20歲的我 / 北某人
```

**驗證**：非空即可。
**Slug 規則**：
- 中文自動轉拼音，用 `-` 連接（「青雲」→ `qing-yun`）
- 英文直接小寫，空格替換為 `-`（「Little North」→ `little-north`）

---

### Q2：基本資訊（可跳過）

```
用一句話介紹一下自己？想到什麼說什麼。

可以包括：
- 年齡 / 年齡層
- 職業 / 身份
- 所在城市
- 學歷背景

範例：
  「25 歲，網路業產品經理，台北」
  「碩二學生，資工系，新竹」
  「自由工作者，插畫師，台中」

跳過也沒關係，直接按 Enter。
```

**解析欄位**：
- `age`：年齡/年齡層
- `occupation`：職業/身份
- `city`：城市
- `education`：學歷背景

---

### Q3：自我畫像（可跳過）

```
最後一個：用一句話描述你自己？

可以包括：
- MBTI / 星座
- 個性特點
- 別人（或你自己）對你的典型評價
- 你覺得自己最特別的地方

範例：
  「INTJ 摩羯座 社恐但話多 深夜emo型選手」
  「ENFP 天秤座 社交蝴蝶 但常常覺得空虛」
  「不知道MBTI，但我很拖延，對有興趣的事又會 obsess」

跳過也行。
```

**解析欄位**：
- `mbti`：MBTI 類型
- `zodiac`：星座
- `personality`：個性標籤列表
- `impression`：主觀印象/自我評價

---

## 彙總確認

```
好的，我整理一下：

  代號：{name}
  基本資訊：{summary}
  自我畫像：{personality_summary}

這樣對嗎？確認後我們進入下一步（匯入原材料）。
```
